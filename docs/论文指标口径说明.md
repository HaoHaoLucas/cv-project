## 2. VG（RefCOCO / RefCOCO+ / RefCOCOg）

### 2.1 论文 Table 5 三列（勿混读）

| 设定 | 预训练数据 | RefCOCO val @1 示例 | 本仓库是否覆盖 |
|------|------------|---------------------|----------------|
| **零样本** | O365, GoldG, Cap4M | **50.41** | **是**（推理 only） |
| + RefC 数据 | + RefCOCO/+/g 训练集 | ~74 | 否 |
| **+ RefC 微调** | 同上 + 在 RefC 上 finetune | **89.19** | 否（需训练） |

**常见误读**：将本仓库 Acc@0.5（~51%）与 **89.19%** 对比——后者为 **微调列**，不是零样本。

### 2.2 @1 / @5 / @10 含义（MMDet 等同表）

- **@1（P@1）**：按置信度排序的 **top-1** 预测框，IoU≥0.5 即命中。与本仓库 **Acc@0.5**（单框）对齐。
- **@5 / @10**：top-5 / top-10 框中是否命中（论文表中 89/94 等 **不是** Acc@0.5）。
